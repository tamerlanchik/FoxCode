Здесь хранится приложение для Android.

Все полезные файлы каждого проекта находятся в %project_name%/app/src/main/
res/ - файлы стилей, картинки и тд - ресурсы.
/java - код Java
/native - код на плюсах.

Будет создано несколько Android-проектов. 
* FoxMap-Java_1 - чисто на Java для разработки и отладки GUI.
